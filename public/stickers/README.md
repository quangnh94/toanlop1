# 📁 Thư mục Sticker

Đây là nơi bạn lưu trữ hình ảnh sticker cho ứng dụng.

## 📝 Hướng dẫn:

1. **Đặt tên file ảnh theo định dạng:** `{id}.png` hoặc `{id}.jpg`
   - Ví dụ: `cinnamoroll-1.png`, `tom-1.jpg`, `pikachu.png`

2. **Định dạng khuyến nghị:**
   - PNG (nền trong suốt) - Khuyến nghị
   - JPG (nền trắng)
   - Kích thước: 200x200 đến 500x500 pixels

## 🔗 Nguồn tìm ảnh miễn phí:

- **Google Images**: Tìm kiếm "cinnamoroll transparent png"
- **Pinterest**: Tìm sticker anime/cartoon
- **Sanrio.com**: Ảnh chính thức
- **Pokemon.com**: Ảnh Pokemon chính thức
- **DeviantArt**: Fan art (kiểm tra bản quyền)

## ⚠️ Lưu ý:

- Sử dụng ảnh có **nền trong suốt** (transparent PNG) để đẹp nhất
- Kiểm tra **bản quyền** khi sử dụng ảnh từ internet
- Đặt tên file **chính xác** với `id` trong file `data/stickers.json`
